# Competitive Programming

Solutions to competitive programming problems, written in C++, organized by topic. Built while preparing for the **SBC Programming Marathon** (ICPC South America — Brazil).

Progress tracked on [Beecrowd](https://www.beecrowd.com.br/) — problems range from foundational logic to graphs and dynamic programming.

---

## Folder structure

```
competitive-programming/
├── 01-Beginners/        # I/O, arithmetic, conditionals, loops
├── 02-arrays-strings/     # Vectors, matrices, string manipulation
├── 03-math/               # Number theory, GCD/LCM, modular arithmetic
├── 04-sorting-search/     # Sorting, binary search, STL algorithms
├── 05-data-structures/    # Stack, queue, map, set, union-find
├── 06-recursion/          # Recursion, backtracking, brute force
├── 07-graphs/             # BFS, DFS, shortest paths
└── 08-dynamic-programming # Memoization, DP tables, classic problems
```

Each file is named after the problem ID and title, e.g. `1001-extremely-basic.cpp`.

---
